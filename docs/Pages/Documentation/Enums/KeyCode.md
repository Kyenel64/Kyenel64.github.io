# KeyCode
<div class = "classInfo">
    Released in v0.1.1-alpha / Last updated in v0.1.1-alpha
</div>

## Description
KeyCodes corresponding to physical keys on the keyboard. <br>
Numerical keycodes begin with a "D" (Eg. D0, D1)
