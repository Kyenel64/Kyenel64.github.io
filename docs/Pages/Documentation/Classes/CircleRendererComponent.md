## Description
Renders a 2D circle.

## Properties
| Name        | Type                | Description                                                                     |
|-------------|---------------------|---------------------------------------------------------------------------------|
| `Color`     | `Color`             | Color of the circle.                                                            |
| `Thickness` | `float`             | Inner thickness of the circle. 0 displays nothing and 1 displays a full circle. |
| `Fade`      | `float`             | The blur of the circle.                                                         |
